# 无人超市管理系统 python532

## 项目概述

无人超市管理系统是基于 Flask 框架开发的，旨在通过高效的信息化管理提高无人超市的运营效率和服务质量。

## 技术栈

- Python 3.7
- Flask
- MySQL 5.7
- Vue.js

## 功能模块

### 管理员功能

- 用户管理
- 员工管理
- 商品分类管理
- 超市商品管理
- 商品进货管理
- 供应商管理
- 系统管理
- 订单管理
- 用户资料管理

### 员工功能

- 超市商品操作
- 商品进货操作
- 订单管理
- 用户资料操作

### 用户功能

- 商品浏览
- 购物车管理
- 我的订单
- 个人中心

## 系统角色

- 管理员
- 员工
- 用户

## 运行环境

- 操作系统：兼容 Linux 与 Windows
- Python 环境要求：Python 3.7 或以上版本
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

1. 安装 Python 3.7 或更高版本
2. 安装 Flask 框架
3. 安装 MySQL 5.7 或更高版本，创建数据库
4. 配置数据库连接信息
5. 部署前端 Vue.js 代码
6. 运行后端 Flask 应用

## 目录结构

```
project/
│
├── app/                 # Flask 应用目录
│   ├── __init__.py
│   ├── models.py        # 数据模型定义
│   ├── views.py         # 视图处理逻辑
│   ├── templates/       # HTML 模板目录
│   └── static/          # 静态文件目录
│
├── config.py            # 配置文件
├── requirements.txt     # 依赖列表
└── run.py               # 应用启动脚本
```

- 基于成熟的 Flask 框架，易于维护和扩展

- 完善的商品与用户管理，提高超市运营效率
